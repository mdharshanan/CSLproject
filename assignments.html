Submission Guide

Use this common format for assignment submission unless your teacher gives different instructions.

File Name

Use course name, student name and assignment title in the file or folder name.

Format

Submit code folders, PDF reports, screenshots or dashboard files based on the task.

Review

Check spelling, links, layout, calculations and required screenshots before submission.

Feedback

Teachers review work and share marks, corrections and improvement comments.
